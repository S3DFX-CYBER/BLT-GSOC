BLT: NetGuardian — Zero-trust finding ingestion, CVE-aware triage, and verified events

GSoC

NetGuardian is a zero-trust ingestion and triage pipeline that connects distributed security producers to BLT. A Cloudflare Python Worker verifies signed ztr-finding-1 envelopes using JCS and HMAC, enforces replay protection and RBAC, stores metadata in D1, keeps encrypted evidence in R2, and enriches findings through BLT-API for CVE-aware scoring and issue creation. An events outbox delivers to downstream consumers; operators review via a static triage UI. Deliverables include versioned ingestion APIs with OpenAPI, end-to-end swarm dataflow, observability and rollout gates with SLOs, and a midterm slice for performance and correctness validation.

BLT University: Interactive Security Labs with Vulnerability Insights

GSoC

An interactive security learning platform on OWASP BLT that turns labs into hands-on, code-driven exercises. Django-based labs migrate to a static Jekyll workflow with Markdown content. Learners follow Identify → Explain → Fix on real vulnerable code. GitHub Actions validate and sync content; a security intelligence pipeline pulls patterns from issues, PRs, and reviews into dashboards and recommendations linking real findings to labs—building practical security mindset and healthier contributions.

Build a cybersecurity guardian that protects a person's digital life in real time (BLT-Vanish — Universal Identity Shield)

GSoC

Evolves BLT-Vanish into a proactive identity defense system: AI-assisted phishing detection, credential hygiene, and correlated suspicious logins. The Universal Identity Shield uses a hybrid local-first and edge architecture for low-latency, explainable risk scoring—phases cover a foundational risk engine, core workflows (dark web monitoring, guided password rotation), and production hardening with Flutter and Python Cloudflare Workers. Deliverables include phishing detection, a zero-plaintext encrypted vault, anomaly correlation, and full documentation.
